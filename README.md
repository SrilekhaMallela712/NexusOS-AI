# 🚀 NexusOS AI

NexusOS AI is a modern AI-powered Student Productivity Dashboard designed to help students manage their academic journey efficiently. It combines task management, analytics, goal tracking, achievement systems, and productivity monitoring in a single platform.

---

## 📌 Features

### 🏠 Landing Page
- Modern responsive design
- Hero section
- Feature showcase
- Statistics dashboard
- Navigation to all modules

### 📊 Analytics Dashboard
- Weekly study analytics
- Skill progress tracking
- Goal completion reports
- Interactive charts using Chart.js

### 📋 Task Management
- Trello-style Kanban Board
- Add tasks dynamically
- Drag and drop functionality
- Local storage support

### 🎯 Goals & Achievements
- Daily goals tracking
- Weekly goals tracking
- XP reward system
- Study streak tracker
- Achievement badges

### 👤 User Profile
- Student profile overview
- Skills section
- Study group members
- Productivity statistics

### ⚙ Settings
- Dark mode settings
- Notification preferences
- Data backup options
- Reset progress functionality

## 🚀 How to Run

### Method 1

Open:

```text
index.html
```

in your browser.

### Method 2 (Recommended)

Open Command Prompt:

```cmd
cd "C:\Users\admin\Desktop\NexusOS AI"
py -m http.server 8000
```

Open:

```text
http://localhost:8000
```

in your browser.
